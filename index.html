Vision Transformer Embedded Video Anomaly Detection using Attention Driven Recurrence


Ummay Maria Muna, Shanta Biswas, Syed Abu Ammar Muhammad Zarif, Tauseef Tajwar, Swakkhar Shatabda
Image and Vision Computing, (IVC), 2024

Developed a model utilizing a custom window-based SRU++ model to detect anomalous behavior from CCTV footage. Model achieved state-of-the-art performance on the UCF Crime, RWF-2K, and SCVD datasets. Under review at Image and Vision Computing (IVC).

Semantic Segmentation using Implicit Graph Neural Networks


Role: Project Lead

Introduced a feasible graph representation of images via dividing images into patches (nodes) and connecting related patches via an edge based on their K Nearest Neighbors. A well-posed function is used to represent the graph neural network which is solved implicitly to perform semantic segmentation in constant memory. Model showed competitive performance as the current state-of-the-art on a small subset of CityScapes.

Supervisor:

Dr. Moshiur Farazi
